# Super Weird One Bud

Rottnest Island surf conditions tracker built with Streamlit.

## Quick Start

```powershell
.\run_local.ps1
```

App opens at http://localhost:8501

## Documentation

See **SETUP.md** for:
- Installation & configuration
- AWS credentials setup
- Deployment guide
- Troubleshooting

## Features

- Track surf sessions with detailed conditions (swell, wind, tide)
- Smart scoring system
- Filter by zone and break
- Cloud storage with AWS S3

## Tech Stack

- Streamlit 1.50.0
- Pandas 2.3.3
- AWS S3 (boto3)
- Python 3.10
